# Desafio MiniWiki
Este projeto, intitulado “Desafio MiniWiki”, é uma demonstração de habilidades de desenvolvimento web usando HTML e semântica. Ele foi criado no CodeSandbox, uma plataforma online para desenvolvimento web.
# Desafio do Projeto VAI NA WEB.

# Sobre o Projeto
<ul> 
  
 <li>O objetivo deste projeto é criar uma mini enciclopédia ou “MiniWiki” usando HTML. O foco principal está em aprender e aplicar corretamente a semântica do HTML.</li>

<li>Estrutura básica: Você criou uma página com a estrutura básica completa usando HTML.</li>

<li>Tags semânticas: Você utilizou as tags semânticas Header, Main e Footer para estruturar o conteúdo da página.</li>

<li>Título da aba: Você colocou seu nome como título da aba do navegador.</li>

<li>Título principal: Você colocou seu nome como o título principal na sua header.</li>

<li>Logo ou foto: Você adicionou uma logo ou uma foto dentro da sua header.</li>

<li>Parágrafo sobre você: Na sua Main, você criou um parágrafo nos contando sobre você, idade, onde mora e outras informações pessoais.</li>

<li>Link com a história do seu bairro: Você inseriu um link na sua Main com a história do seu bairro.</li>

<li>Subtítulo “Suas Características”: Você criou um subtítulo “Suas Características” e uma lista não ordenada com as respostas.</li>

<li>Subtítulo “Suas Habilidades”: Você criou um subtítulo “Suas Habilidades” e uma lista ordenada com as respostas.</li>

<li>Subtítulo “Comidas preferidas”: Você criou um subtítulo “Comidas preferidas” e uma lista ordenada com as respostas.</li>

<li>Redes Sociais: No seu footer, você adicionou suas Redes Sociais.</li>

 </ul>
 
# Tecnologias Utilizadas

<img src="https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white" alt=" html" img>
Usado para criar a estrutura básica da página web. Isso inclui a criação de diferentes seções na página, como cabeçalhos, parágrafos, listas e links. A semântica do HTML é usada para garantir que a estrutura da página seja compreensível tanto para os usuários quanto para os motores de busca.
